# Movie Search App

## Описание
Простой фронтенд-проект на React + TypeScript, который позволяет искать фильмы через OMDb API,
показывать результаты в виде карточек и открывать подробную информацию в модальном окне.
Основной акцент — логика взаимодействия с API и автокомплит.

## Функционал

- Поиск фильмов по названию
- Автодополнение (suggestions) при вводе текста
- Дебаунс запросов к API
- Пагинация: подгрузка фильмов при прокрутке (IntersectionObserver)
- Модальное окно с подробной информацией о фильме
- Обработка ошибок постеров (если нет изображения, подставляется заглушка)
- Закрытие автокомплита и модалки при клике вне элемента

## Технологии

- React + TypeScript
- Tailwind CSS v3
- OMDb API
- IntersectionObserver для бесконечной прокрутки
- Дебаунс для оптимизации запросов

## Установка и запуск
```
git clone https://github.com/petgear/OMDB-app
cd movie-search-app
npm install
npm run dev
```

## Настройка API ключа

Для работы нужен ключ OMDb API (можно получить бесплатно на OMDb API)
Заменить значение apikey в AutoComplete.tsx:
```ts
const url = `https://www.omdbapi.com/?apikey=<YOUR_API_KEY>&s=${query}&type=movie`;
```


## Как использовать

- Вводите название фильма в поле поиска.
- Появятся предложения (suggestions), выбираете один или нажимаете "Поиск".
- Карточки фильмов отображаются ниже.
- Клик на карточку открывает модальное окно с деталями фильма.
- Клик вне модалки закрывает её.
- При прокрутке списка вниз подгружаются новые фильмы (если они есть)
